# FiveDesigner Agent 模式设计文档

## 一、核心定位升级

### 当前模式（被动执行）
- 用户手动选择：模型、尺寸、参考图
- 用户手动填写：提示词、参数
- 用户手动触发：生成、保存

### Agent 模式（智能助理）
- 用户只说目标："做一套汉服电商海报"
- AI 自动完成：理解 → 规划 → 执行 → 优化

## 二、四大核心模块

### 1. 需求解析引擎（Intent Parser）

**输入**：自然语言描述
```
"帮我做一个古风仙侠短视频，女主在桃花林舞剑，氛围感拉满"
```

**输出**：结构化创作需求
```typescript
{
  type: 'video',           // 创作类型：image/video/audio
  style: '古风仙侠',        // 风格
  subject: '女主',          // 主体
  scene: '桃花林',          // 场景
  action: '舞剑',           // 动作
  mood: '氛围感、飘逸、唯美', // 情绪
  duration: 5,             // 时长（视频）
  aspectRatio: '9:16',     // 比例
  outputCount: 4           // 输出数量
}
```

**实现**：
- 使用 LLM（如 GPT-4/Claude）解析用户意图
- 专项优化：中文文化、古风、亚洲人脸等场景
- 自动补全缺失参数

### 2. 任务自主拆解（Task Planner）

**输入**：结构化需求

**输出**：任务依赖图
```typescript
[
  { id: 'char_design', name: '设计女主形象', deps: [], type: 'image' },
  { id: 'scene_design', name: '设计桃花林场景', deps: [], type: 'image' },
  { id: 'storyboard_1', name: '分镜1-起势', deps: ['char_design', 'scene_design'], type: 'image' },
  { id: 'storyboard_2', name: '分镜2-挥剑', deps: ['storyboard_1'], type: 'image' },
  { id: 'storyboard_3', name: '分镜3-转身', deps: ['storyboard_2'], type: 'image' },
  { id: 'storyboard_4', name: '分镜4-落剑', deps: ['storyboard_3'], type: 'image' },
  { id: 'video_gen', name: '合成视频', deps: ['storyboard_1', 'storyboard_2', 'storyboard_3', 'storyboard_4'], type: 'video' }
]
```

**实现**：
- 基于项目数据（角色、场景、分镜）自动规划
- 支持并行任务（无依赖的任务同时执行）
- 支持串行任务（有依赖的任务按顺序执行）

### 3. 工具链智能调用（Tool Orchestrator）

**自动选择模型**：
| 任务类型 | 推荐模型 | 理由 |
|---------|---------|------|
| 角色设计 | Seedream/Flux | 人物一致性好 |
| 场景设计 | Kling Image | 场景细节丰富 |
| 视频生成 | Seedance/Kling Video | 动作流畅 |
| 配音 | MiniMax TTS | 中文自然 |

**自动匹配参数**：
- 根据风格自动选择：比例、尺寸、质量
- 根据上下文自动继承：角色形象、场景风格
- 自动优化提示词：添加风格词、质量词

### 4. 闭环校验与优化（Quality Validator）

**风格一致性检查**：
- 角色形象是否统一（人脸相似度）
- 场景色调是否一致（色彩分析）
- 分镜构图是否连贯（视觉连贯性）

**自动修复策略**：
- 不一致 → 使用参考图重新生成
- 质量低 → 提高质量参数重试
- 崩坏 → 更换模型或调整提示词

**多版本并行**：
- 默认生成 4 个方案供选择
- 支持局部修改（"把剑换成扇子"）

## 三、工作流程

```
用户输入 → 需求解析 → 任务规划 → 并行/串行执行 → 质量校验 → 输出结果
              ↓              ↓              ↓              ↓
           意图识别      生成任务图      调用工具链      自动修复
```

## 八、示例场景

### 场景1：电商海报
**用户输入**："做一套汉服电商海报，包含模特展示、细节特写、场景搭配"

**Agent 执行**：
1. 解析：3张海报（模特图、细节图、场景图）
2. 规划：先生成角色 → 再生成场景 → 最后合成海报
3. 执行：并行生成3张，保持风格一致
4. 输出：4套方案供选择

### 场景2：短视频
**用户输入**："做一个古风舞剑短视频，10秒"

**Agent 执行**：
1. 解析：视频、古风、舞剑、10秒
2. 规划：角色设计 → 场景设计 → 4个分镜 → 视频合成
3. 执行：串行生成，确保动作连贯
4. 输出：成品视频 + 分镜图

### 场景3：角色一致性
**用户输入**："给这个角色生成不同表情和动作"

**Agent 执行**：
1. 解析：图生图、多姿态、保持一致性
2. 规划：使用参考图模式，批量生成
3. 执行：并行生成8张不同姿态
4. 输出：角色姿态库
