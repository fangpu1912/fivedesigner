# 样片审阅页面开发计划

## 需求概述
创建一个样片审阅页面，支持：
1. 将分镜、视频、音频、字幕等元素按时间轴排列
2. 可视化编辑（拖拽调整顺序、时长）
3. 实时预览合成效果
4. 合并成样片并保存
5. 导出到剪映成为草稿

## 技术调研

### 需要检查的现有功能
- [ ] 分镜数据结构（storyboard）
- [ ] 配音数据结构（dubbing）
- [ ] 视频生成状态存储
- [ ] 现有的时间轴/序列组件
- [ ] 剪映API/导入格式

### 剪映草稿格式研究
剪映支持导入的格式：
- 剪映专业版支持导入 JSON 格式的草稿
- 需要研究剪映草稿的 JSON 结构
- 需要映射：视频轨道、音频轨道、字幕轨道

## 数据结构定义

### 样片项目 (SampleProject)
```typescript
interface SampleProject {
  id: string;
  episode_id: string;
  name: string;
  duration: number; // 总时长(秒)
  tracks: Track[];
  created_at: string;
  updated_at: string;
}

interface Track {
  id: string;
  type: 'video' | 'audio' | 'subtitle' | 'effect';
  name: string;
  items: TrackItem[];
  isVisible: boolean;
  isLocked: boolean;
}

interface TrackItem {
  id: string;
  type: 'storyboard' | 'dubbing' | 'music' | 'subtitle' | 'transition';
  source_id: string; // 关联的分镜/配音ID
  start_time: number; // 在时间轴上的开始时间(秒)
  duration: number; // 时长(秒)
  // 视频特有
  video_url?: string;
  // 音频特有
  audio_url?: string;
  volume?: number;
  // 字幕特有
  text?: string;
  subtitle_style?: SubtitleStyle;
  // 转场特有
  transition_type?: string;
  transition_duration?: number;
}

interface SubtitleStyle {
  font_family: string;
  font_size: number;
  color: string;
  background_color?: string;
  position: 'top' | 'middle' | 'bottom';
}
```

## 技术难点

### 1. 时间轴性能优化
- 虚拟滚动（处理大量片段）
- Canvas渲染优化
- 拖拽性能

### 2. 视频合成
- 浏览器端合成（WebCodecs API）
- 或使用FFmpeg.wasm
- 或使用后端服务

### 3. 剪映格式兼容
- 需要逆向研究剪映草稿格式
- 可能需要多次迭代测试

## 风险评估
1. **剪映导出**: 格式可能不公开，需要研究或联系剪映团队
2. **视频合成**: 浏览器端性能有限，大视频可能需要后端支持
3. **时间轴复杂度**: 需要仔细设计状态管理避免性能问题
